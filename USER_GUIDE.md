# User Guide

## Getting Started

### Creating an Account

1. Navigate to the home page
2. Click "Sign Up" or "Get Started"
3. Fill in your information:
   - Name
   - Email address
   - Password (minimum 6 characters)
   - Confirm password
4. Click "Sign Up"
5. You'll be automatically logged in and redirected to the dashboard

### Logging In

1. Click "Login" in the navigation bar
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to your dashboard

## Dashboard

The dashboard provides an overview of your tasks:

- **Statistics**: View counts of tasks by status (To Do, In Progress, Completed)
- **Recent Tasks**: See your 5 most recent tasks
- **Quick Actions**: Create a new task directly from the dashboard

## Managing Tasks

### Creating a Task

1. Click "Create New Task" from the dashboard or tasks page
2. Fill in the task details:
   - **Title** (required): A brief description of the task
   - **Description** (optional): Detailed information about the task
   - **Status**: Choose from To Do, In Progress, or Completed
   - **Priority**: Set as Low, Medium, or High
   - **Due Date** (optional): Set a deadline for the task
   - **Assign To** (optional): Assign the task to a team member
   - **Tags** (optional): Add comma-separated tags for categorization
3. Click "Create Task"

### Viewing Tasks

1. Navigate to the "Tasks" page from the navigation bar
2. View all your tasks in a grid layout
3. Click on any task to view its details

### Editing a Task

1. Open the task you want to edit
2. Click the "Edit" button
3. Modify the task details
4. Click "Update Task"

### Deleting a Task

1. Open the task you want to delete
2. Click the "Delete" button
3. Confirm the deletion

### Filtering Tasks

On the Tasks page, you can filter tasks by:

- **Search**: Search by title or description
- **Status**: Filter by To Do, In Progress, or Completed
- **Priority**: Filter by Low, Medium, or High

## Task Details

When viewing a task, you'll see:

- Full task title and description
- Current status and priority
- Due date (if set)
- Assigned user (if assigned)
- Creator information
- Tags (if any)
- Creation date

## Real-time Updates

The application uses real-time updates, so you'll automatically see:

- New tasks assigned to you
- Updates to tasks you're watching
- Status changes
- Priority changes

## User Profile

### Updating Your Profile

1. Click on your name in the navigation bar
2. Update your name or email
3. Click "Update Profile"

Note: Your role cannot be changed (only admins can modify roles)

## Tips for Best Results

1. **Use Clear Titles**: Make task titles descriptive and specific
2. **Set Priorities**: Use priority levels to focus on important tasks
3. **Set Due Dates**: Keep track of deadlines
4. **Use Tags**: Organize tasks with relevant tags
5. **Assign Tasks**: Delegate work to team members
6. **Update Status**: Keep task status current for accurate tracking

## Keyboard Shortcuts

- `Ctrl/Cmd + K`: Quick search (future feature)
- `Esc`: Close modals (future feature)

## Troubleshooting

### Can't log in?
- Verify your email and password are correct
- Make sure you've registered an account
- Try resetting your password (if feature is available)

### Tasks not showing?
- Check your filters
- Verify you're logged in
- Refresh the page

### Real-time updates not working?
- Check your internet connection
- Refresh the page
- Log out and log back in
